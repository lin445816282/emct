### 2.1 当前 AI 能力盘点

| 功能 | 实现 | 评分 |

| 波动率仓位调整 | ✅ vol_factor | ⭐⭐⭐ |

### 2.2 🔴 AI 优化升级项

#### 🔴 1. 缺少盘中实时信号预警
- **现状**：信号扫描依赖 cron（15:00）和手动点击，盘中无自动触发
- **优化**：增加盘中定时扫描（如每30分钟），配合 `market_regime_ok()` 过滤，发现 strong_buy/strong_sell 时推送通知
- **收益**：AI 从「收盘后批量决策」升级为「盘中动态监控」

#### 🔴 2. DeepSeek AI 复盘过于被动
- **现状**：需手动点「AI分析」按钮，分析结果纯文本
- **优化**：
  - 每日收盘后自动调 DeepSeek 生成复盘报告，cron 推送
  - 在信号卡片上增加「AI 解读」按钮，对单只股票做深度分析
  - 复盘结果结构化存储（加 `ai_review` 表），长期追踪 AI 判断准确率

#### 🔴 3. 因子权重固化，无自适应
- **现状**：6因子权重硬编码（25/20/15/15/10/15%）
- **优化**：
  - 回测引擎增加「网格搜索权重」功能 → 自动找最优权重组合
  - 引入滚动窗口优化（最近 N 天数据重新校准权重）
  - 因子有效性衰减（长期无效的因子自动降权）

#### 🔴 4. 缺少 AI 自然语言交互入口
- **现状**：所有操作必须点按钮
- **优化**：增加一个 AI 对话框（如底部浮动按钮），支持自然语言查询：
  - "今天哪些股票值得关注？"
  - "我的持仓风险如何？"
  - "帮我回测最近一个月的策略"
  - 对接 DeepSeek API，后端封装查询函数

#### 🟡 5. 回测 vs 实盘偏离无监控
- **现状**：回测独立运行，不对比实盘
- **优化**：加「回测vs实盘偏离度」指标，AI 自动检测策略是否失效

#### 🟡 6. 无持仓组合优化建议
- **现状**：持仓页只展示数据
- **优化**：AI 分析持仓集中度/板块风险/波动率，给出调整建议

| 模拟页 | 账户+回测+基准对比 | 实时数据区无操作 |

### 3.2 🔴 真人使用优化项

#### 🔴 1. 信号页布局混乱
- **现状**：数据面板（可折叠）+ 股票池管理 + 信号卡片，堆在同一页
- **优化**：
  - 股票池管理移到独立 Tab 或设置页
  - 数据面板默认折叠，仅显示一行摘要（如"📊 20只 · 1700条 · 最新 07-17"）
  - 信号卡片上方加「快速筛选」：全部/强烈买入/买入/卖出

#### 🔴 2. 缺少 Dashboard 首页
- **现状**：打开即信号页，无全局视图
- **优化**：新增「总览」Tab，一屏展示：
  - 账户总资产卡片
  - 今日 P&L / 持仓市值 / 可用现金
  - 今日信号摘要（几买几卖）
  - 最近成交
  - 风险概览

#### 🔴 3. 持仓页信息密度低
- **现状**：每只持仓一张大卡片
- **优化**：
  - 顶部加汇总卡片（总市值/总盈亏/持仓数）
  - 持仓列表改为紧凑行（代码/名称/股数/成本/现价/盈亏%）
  - 盈亏排序，浮亏置顶

#### 🔴 4. 「一键跟单」操作风险高
- **现状**：一个按钮给所有 pending 信号下单，无预览
- **优化**：
  - 点击后弹出预览面板：列出每只要买的股票+金额+理由
  - 支持勾选/取消单只
  - 二次确认后再执行

#### 🟡 5. 模拟/实盘切换不明显
- **现状**：右上角小开关 + 小字
- **优化**：顶部栏背景色区分（蓝色=模拟，红色=实盘），加醒目标签

#### 🟡 6. 缺少移动端适配
- **现状**：Vant4 组件库本身支持移动端，但布局未针对小屏优化
- **优化**：
  - 信号卡片在小屏上单列
  - 数据面板在小屏上更紧凑
  - 底部 Tab 固定

#### 🟡 7. 缺少快捷操作
- **优化**：
  - 模拟页增加「快捷买入」浮动按钮
  - 信号卡片长按出菜单（下单/忽略/详情）
  - 持仓卡片左滑出「卖出」按钮

## 五、AI 使用手册

### 5.1 AI 能做什么

#### 信号分析
```
POST /api/signals/scan          → 全量扫描20只股票，生成今日信号
GET  /api/signals               → 读取历史信号（支持 ?status=pending 过滤）
GET  /api/signals/analyze/{code} → 单股实时多因子分析
```

#### 模拟交易
```
POST /api/sim/quick-trade       → 快速买入/卖出（自动取日线收盘价）
POST /api/sim/auto-trade        → 自动跟单：基于今日信号开仓
POST /api/sim/settle-t1         → 退出检查：止损/止盈/到期/信号反转
POST /api/sim/auto-check        → 实时止损止盈扫描
GET  /api/sim/account           → 账户总览
GET  /api/sim/positions         → 持仓列表
GET  /api/sim/equity-curve      → 净值曲线
```

#### 回测
```
GET  /api/sim/backtest          → 基于历史信号回测（summary + trades + equity）
POST /api/sim/sync-benchmark    → 同步沪深300基准
GET  /api/sim/benchmark         → 获取基准数据
```

#### 风控
```
GET  /api/orders/risk/summary   → 风控总览（仓位/集中度/止损止盈）
GET  /api/sim/market-regime     → 市场趋势判断（是否适合开仓）
```

#### AI 复盘
```
GET  /api/review/analyze        → DeepSeek 生成复盘分析
GET  /api/review/summary        → 复盘摘要（供 AI 进一步分析）
```

### 5.2 典型 AI 操作流程

**每日收盘自动化链路（cron）：**
```
15:00 → POST /api/signals/scan         # 扫描信号
15:05 → POST /api/sim/settle-t1         # 退出检查（平仓触发项）
      → POST /api/data/sync?days=1      # 同步日线
15:06 → POST /api/sim/auto-trade        # 自动跟单开仓
      → GET  /api/review/analyze        # AI 复盘报告
      → 推送微信通知
```

**手动回测验证：**
1. `GET /api/sim/backtest` → 查看策略历史表现
2. 对照 `GET /api/sim/benchmark` 判断是否跑赢沪深300
3. 调整因子权重（`analyzer.py`）→ 重新回测

**风控应急：**
1. `GET /api/orders/risk/summary` → 检查当前风险
2. `POST /api/sim/auto-check` → 强制扫描止损止盈
3. `GET /api/sim/market-regime` → 确认市场是否可以交易

## 六、真人使用手册

### 6.1 快速入门

#### 第一次使用
1. 打开 https://www.ct256.cn/emct/
2. 确认右上角为「模拟」模式（蓝色开关）
3. 点「📊 数据状态」→ 「同步日线」（拉取60天K线）
4. 点「分析扫描」→ 生成今日信号
5. 浏览信号卡片，展开感兴趣的股票看理由标签

#### 模拟交易
1. 在信号卡片上点「下单」→ 自动买入100股（成交价=最新收盘价）
2. 切到「持仓」Tab 查看持仓，点「卖出」可平仓
3. 切到「复盘」Tab → 「AI分析」看 DeepSeek 交易建议
4. 切到「模拟」Tab → 「运行回测」验证策略历史表现

#### 实盘交易
1. 开启 Edge CDP：`msedge.exe --remote-debugging-port=9222`
2. 登录东方财富交易页面
3. 切换右上角开关为「实盘」
4. 在信号卡片上点「下单」→ 创建订单
5. 切到「订单」Tab → 「执行」发送到东方财富

### 6.2 各 Tab 功能速查

| Tab | 核心功能 | 关键操作 |
|-----|---------|---------|
| 📊 信号 | 多因子评分 + 股票池管理 | 扫描/下单/一键跟单 |
| 💰 持仓 | 模拟持仓 + 盈亏 | 卖出/刷新行情 |
| 📋 订单 | 订单创建 + 执行 | CDP下单/风控检查 |
| 📈 复盘 | 交易统计 + AI分析 | 同步/分析/净值图 |
| 🎯 模拟 | 回测 + 基准对比 | 运行回测/同步基准 |

### 6.3 常见操作

**如何添加新股票？**
1. 信号页 → 展开「📊 数据状态」
2. 点「+添加」→ 填代码/名称/市场(SH或SZ)/板块
3. 点「同步日线」拉取K线数据

**如何回测策略？**
1. 模拟页 → 点「▶ 运行回测」
2. 查看6格绩效卡片（总收益/年化/回撤/胜率/盈亏比/交易数）
3. 净值图对比沪深300基准

**如何查看 AI 建议？**
1. 复盘页 → 先点「同步订单」
2. 再点「AI分析」→ 等待DeepSeek生成

### 6.4 注意事项

- ⚠️ 模拟交易仅在交易日 9:30-15:00 可执行（非交易时间按钮禁用）
- ⚠️ 实盘需 Edge CDP 运行中，否则订单无法执行
- ⚠️ 「一键跟单」会买入所有 pending 信号对应的股票（无预览）
- ⚠️ 因子权重修改在 `analyzer.py`，改动后需重新扫描信号生效
- ⚠️ 数据库路径：`~/projects/emct/backend/backend/data/emct.db`
